## Barème des points

L'attribution des points par série se fait de la manière suivante :

- 20 points si vous trouvez le vainqueur et le finaliste dans le bon ordre
- 12 points si vous trouvez uniquement le vainqueur
- 10 points si vous trouvez le vainqueur et le finaliste aux mauvaises places
- 7 points si vous trouvez uniquement le finaliste
- 5 points si vous trouvez le vainqueur ou le finaliste à la mauvaise place
- 0 point dans tous les autres cas

## Classement et égalité

Le cumul des points marqués pour chaque série de la journée permet d'établir un classement général entre les joueurs. Dans le cas où plusieurs seraient ex-aequo, le pronostic sur la durée du match le plus long permettra de les départager.

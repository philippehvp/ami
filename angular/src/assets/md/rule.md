## Barème des points

Le calcul des points par série suit les règles suivantes.

| Points | Conditions                                  |
| ------ | ------------------------------------------- |
| 20     | Vainqueur et le finaliste dans le bon ordre |
| 12     | Uniquement le vainqueur juste               |
| 10     | Vainqueur et finaliste inversés             |
| 7      | Uniquement le finaliste juste               |
| 5      | Vainqueur ou finaliste à la mauvaise place  |
| 0      | Tous les autres cas                         |

## Classement et égalité

Le cumul des points marqués pour chaque série de la journée permet d'établir un classement général entre les joueurs. Dans le cas où plusieurs d'entre eux seraient ex-aequo, le pronostic sur la durée du match le plus long de la journée permettra de les départager.
